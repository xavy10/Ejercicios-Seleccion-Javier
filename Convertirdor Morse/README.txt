Instrucciones: Para ejecutar la aplicación, nos iremos a la carpeta src, nos iremos a la carpeta modelo y correremos la clase llamada Principal, para ello nos pedirá inmediatamente que ingresemos la cadena que queremos convertir, podemos ingresar una cadena normal como "mi nombre es javier" o directamente podemos ingresar el código morse para que nos genera la cadena normal o la traducción a código morse. La aplicación es capaz de detectar si es una cadena normal o es código morse, por lo cual solo ingresaremos una de las dos, y la aplicación nos generara el resultado, después de mostrar el resultado nos dirá desea convertir otra cadena(S/N), para detener la aplicación, ingresaremos cualquier otra tecla que no sea "S".
